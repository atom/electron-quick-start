Kiosk Installation Guide (raspberry pi)

  • Install the latest Raspbian Buster with Desktop (not with recommended software)
  • Go through setup process
    • Set localization settings

    • Ignore the password (we will be deleting this user anyway)
    • Set overscan if necessary
    • Set wifi if necessary
    • Update software
    • Click "later" on restart
  • Open "Preferences --> Raspberry Pi configuration"
    • Change hostname (if the asset tag is A00111, the hostname would be cdcvicks-00111)
    • Enable SSH
  • You are now ready to run the kiosk installation script
